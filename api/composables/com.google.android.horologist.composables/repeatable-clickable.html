repeatableClickable

fun Modifier.repeatableClickable(interactionSource: MutableInteractionSource, indication: Indication?, enabled: Boolean = true, onClickLabel: String? = null, role: Role? = null, initialDelay: Long = 500, incrementalDelay: Long = 60, onClick: () -> Unit, onRepeatableClick: () -> Unit = onClick): Modifier

This modifier provides functionality to increment or decrement values repeatedly by holding down the composable. Should be used instead of clickable modifier to achieve clickable and repeatable clickable behavior. Can't be used along with clickable modifier as it already implements it.

Code from https://cs.android.com/androidx/platform/frameworks/support/+/androidx-main:wear/compose/compose-material-core/src/main/java/androidx/wear/compose/materialcore/RepeatableClickable.kt

Callbacks onClick and onRepeatableClick are different. onClick is triggered only when the hold duration is shorter than initialDelay and no repeatable clicks happened. onRepeatableClick is repeatedly triggered when the hold duration is longer than initialDelay with incrementalDelay intervals.

Parameters

interactionSource

MutableInteractionSource that will be used to dispatch PressInteraction.Press when this clickable is pressed. Only the initial (first) press will be recorded and dispatched with MutableInteractionSource.

indication

indication to be shown when modified element is pressed. By default, indication from LocalIndication will be used. Pass null to show no indication, or current value from LocalIndication to show theme default

enabled

Controls the enabled state. When false, onClick, and this modifier will appear disabled for accessibility services

onClickLabel

semantic / accessibility label for the onClick action

role

the type of user interface element. Accessibility services might use this to describe the element or do customizations

initialDelay

The initial delay before the click starts repeating, in ms

incrementalDelay

The delay between each repeated click, in ms

onClick

will be called when user clicks on the element

onRepeatableClick

will be called after the initialDelay with incrementalDelay between each call until the touch is released
