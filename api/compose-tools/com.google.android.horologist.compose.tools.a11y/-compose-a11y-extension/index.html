Paparazzi Render Extension that collects Accessibility information for Compose hierarchies.

Currently captures Content Description, State Description, On Click, Role, Disabled, Heading, Custom Actions, Text, and Progress. These are saved as AccessibilityState.Element in the accessibilityState list.
